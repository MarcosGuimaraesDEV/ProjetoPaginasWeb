  • A font-family propriedade define o tipo de letra de um elemento.
  • font-size controla o tamanho do texto exibido.
  • font-weight define como o texto fino ou grosso é exibido.
  • A text-align propriedade coloca o texto à esquerda, à direita ou no centro de seu contêiner pai.
  • O texto pode ter dois atributos de cor diferentes: color e background-color. color define a cor do texto, enquanto background-color define a cor por trás do texto.
  • CSS pode tornar um elemento transparente com a opacitypropriedade.
  • CSS também pode definir o plano de fundo de um elemento para uma imagem com a background-image propriedade.
  • O !important sinalizador substituirá qualquer estilo, mas quase nunca deve ser usado, pois é extremamente difícil de substituir.
