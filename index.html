Estudando Html

  • HTML significa Hyper Text Markup Language e é usado para criar a estrutura e o conteúdo de uma página da web.
  • A maioria dos elementos HTML contém tags de abertura e fechamento com texto bruto ou outras tags HTML entre eles.
  • Elementos HTML podem ser aninhados dentro de outros elementos. O elemento delimitado é o filho do elemento pai delimitador.
  • <body>Qualquer conteúdo visível deve ser colocado dentro das tags de abertura e fechamento.
  • Títulos e subtítulos, <h1>para <h6>tags, são usados para fornecer títulos para seções de conteúdo.
  • <p>e <span>as <div>tags especificam texto ou blocos.
  • As tags <em>e <strong>são usadas para enfatizar o texto.
  • As quebras de linha são criadas com a <br>tag.
  • As listas ordenadas ( <ol>) são numeradas e as não ordenadas ( <ul>) são marcadas.
  • Imagens ( <img>) e vídeos ( <video>) podem ser adicionados vinculando a uma fonte existente.
