Tabelas

  • O <table>elemento cria uma tabela.
  • O <tr>elemento adiciona linhas a uma tabela.
  • Para adicionar dados a uma linha, você pode usar o <td>elemento.
  • Os títulos das tabelas esclarecem o significado dos dados. Os títulos são adicionados com o <th>elemento.
  • Os dados da tabela podem abranger colunas usando o colspan atributo.
  • Os dados da tabela podem abranger linhas usando o rowspan atributo.
  • As tabelas podem ser divididas em três seções principais: um cabeçalho, um corpo e um rodapé.
  • O cabeçalho de uma tabela é criado com o <thead> elemento.
  • O corpo de uma tabela é criado com o <tbody> elemento.
  • O rodapé de uma tabela é criado com o <tfoot>elemento.
  • Parabéns por completar as Tabelas HTML!
