Formulários HTML

  • O objetivo da <form> é permitir que os usuários insiram informações e as enviem.
  • O atributo <form>'s determina para onde vão as informações do formulário.action
  • O atributo <form>'s determina como as informações são enviadas e processadas.method
  • Para adicionar campos para os usuários inserirem informações, usamos o <input>elemento e definimos o type atributo para um campo de nossa escolha:
  • A configuração type para "text" cria um campo de linha única para entrada de texto.
  • A configuração type para "password" cria um campo de linha única que censura a entrada de texto.
  • A configuração type para "number" cria um campo de linha única para entrada de número.
  • A configuração type para "range" cria um controle deslizante para selecionar a partir de um intervalo de números.
  • A configuração type para "checkbox" cria uma única caixa de seleção que pode ser emparelhada com outras caixas de seleção.
  • A configuração type para "radio" cria um botão de opção que pode ser emparelhado com outros botões de opção.
  • A configuração type para "list" emparelhará <input>com um <datalist>elemento se os id dois forem iguais.
  • A configuração type para "submit" cria um botão de envio.
  • Um <select> elemento é preenchido com <option> elementos e renderiza uma seleção de lista suspensa.
  • Um <datalist>elemento é preenchido com <option>elementos e funciona com um <input>para pesquisar opções.
  • Um <textarea> elemento é um campo de entrada de texto que possui uma área personalizável.
  • Quando a <form> é enviado, o namedos campos que aceitam entrada e o value desses campos são enviados como name=value pares.
  • O uso do <form> elemento em conjunto com os outros elementos listados acima nos permite criar sites que levam em consideração os desejos e necessidades de nossos usuários. Aproveite para pegar o que aprendeu e aplicar!
